# Project
    > A Full-Stack E-commerce Website 

## To run:

Create Virtual Environment:
    -- pip install pipenv
    -- pipenv shell

Install Packages:
    -- pip install -r requirements.txt

Create Superuser
    -- py manage.py createsuperuser

Connect Database:
    > Open phpmyadmin and create a DB called "shopkartdb"
